# 💻 Lista de Exercícios - Algoritmos em Python

Este repositório contém a resolução de exercícios da disciplina de **Algoritmos**, com foco no desenvolvimento do raciocínio lógico e na prática de programação utilizando Python.

---

## 📚 Conteúdos Trabalhados

Os exercícios foram resolvidos utilizando apenas os conceitos estudados até o momento:

- Variáveis
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`while`, `for`)

---

## 🧠 Objetivo

O objetivo deste projeto é:

- Praticar lógica de programação
- Desenvolver autonomia na resolução de problemas
- Consolidar conceitos básicos de algoritmos
- Preparar para avaliações da disciplina
